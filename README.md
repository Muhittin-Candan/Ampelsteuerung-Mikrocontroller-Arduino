# Ampelsteuerung-Mikrocontroller-
Ampelsteuerung Mikrocontroller  simuliert mittels Arduino Board 

-------
Aufgabenstellung
    1. Aufbau einer Ampelsteuerung für Autofahrer und Fußgänger
    2. Erweiterung der Schaltung durch ein Taster für die Fußgänger
    3. Erweiterung der Schaltung durch Lichtsteuerung für den Nachtbetrieb 
	
	---------

    Arbeitsmittel
    1. Einen funktionsfähigen Arbeitsplatz mit den benötigten Programm (Arduino.exe)
    2. Ein funktionsfähigen Arduino mit Zubehör (Kastennummer 39)
    3. Bauelemente siehe Steckplan
    (5x 220 Widerstände, 2x Grüne LEDs, 2x Rote LEDs, 1x Gelbes LED, 1x Taster, 
	2x 1k Wiederstände, 1x LDR/lichtempfindlicher elektrischer Widerstand)
